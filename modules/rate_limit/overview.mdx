## Installation

<Steps>
<Step title="Update config">
Add the following to your `backend.yaml`:

```yaml
modules:
  rate_limit: {}

```

</Step>
<Step title="Call scripts">
<CodeGroup>

```bash curl
curl --request POST \
    --url https://dog-api.kinduff.com/api/facts
```

```typescript TypeScript
let response = await fetch("https://dog-api.kinduff.com/api/facts", {});
```

</CodeGroup>
</Step>
</Steps>

## Scripts

<CardGroup><Card title="Throttle" href="/modules/rate_limit/scripts/throttle">Limit the amount of times an request can be made by a given key.</Card>
<Card title="Throttle Public" href="/modules/rate_limit/scripts/throttle_public">Limit the amount of times a public request can be made by a given key. This will rate limit based off the user's IP address.</Card></CardGroup>
