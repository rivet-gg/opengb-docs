## Installation

<Steps>
<Step title="Update config">
Add the following to your `backend.yaml`:

```yaml
modules:
  users: {}

```

</Step>
<Step title="Call scripts">
<CodeGroup>

```bash curl
curl --request POST \
    --url https://dog-api.kinduff.com/api/facts
```

```typescript TypeScript
let response = await fetch("https://dog-api.kinduff.com/api/facts", {});
```

</CodeGroup>
</Step>
