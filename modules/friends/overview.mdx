## Installation

<Steps>
<Step title="Update config">
Add the following to your `backend.yaml`:

```yaml
modules:
  friends: {}

```

</Step>
<Step title="Call scripts">
<CodeGroup>

```bash curl
curl --request POST \
    --url https://dog-api.kinduff.com/api/facts
```

```typescript TypeScript
let response = await fetch("https://dog-api.kinduff.com/api/facts", {});
```

</CodeGroup>
</Step>
</Steps>

- **Friend Request Already Exists** (`friend_request_already_exists`) 
- **Friend Request Not Found** (`friend_request_not_found`) 
- **Not Friend Request Recipient** (`not_friend_request_recipient`)
